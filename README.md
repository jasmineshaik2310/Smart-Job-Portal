#Smart Job Portal (Resume Matching System)

Smart Job Portal is a web application that matches uploaded resumes with available job listings and shows a match percentage for each job based on skills and keywords.

##What it does

- Users can upload their resume  
- Extracts skills and keywords from resume  
- Compares resume with job descriptions  
- Calculates match percentage for each job  
- Suggests most relevant jobs to the user  
- Displays results in a simple and clean UI  

##Technologies Used

- HTML  
- CSS  
- JavaScript
- python
